## Typography
- **Primary Typeface**: Mundial
  - Available at: https://tipotype.com/mundial
  - Adobe Fonts: https://fonts.adobe.com
  - Weights: Hair, Thin, Light, Regular, Demibold, Bold, Black
  - Use Regular (400) for body text
  - Use Demibold (600) for UI elements
  - Use Bold (700) for headings
  - Use Black (900) for hero titles

## Spacing Scale
- Base: 4px (0.25rem)
- Scale: 4, 8, 12, 16, 24, 32, 48, 64px

## Grid & Layout
- Container maximum width: 1200px
- Horizontal padding: 16px (mobile), 24px (desktop)
- Grid columns: 1 (mobile), 12 (desktop)
- Grid gap: 16px (mobile), 24px (desktop)

## Border Radius
- Small: 8px (buttons, tags)
- Default: 16px (cards, containers)
- Large: 24px (hero sections)
- Full: 9999px (pills, badges)

## Shadows
- Default: 0 8px 24px rgba(112, 34, 131, 0.12)
- Large: 0 16px 48px rgba(112, 34, 131, 0.16)

## Logo Usage
- Minimum clear space: Height of "D" character
- Two versions: Large-scale (>10cm) and Small-scale (<10cm)
- Shell symbol can be used in isolation within circular seal
- Never stretch, compress, or distort
- Never rotate or tilt
- Never add borders or decorative elements
- Never use unapproved colors
- Never alter or remove logo elements

## Voice & Tone
- Modern, accessible, and inclusive
- Direct and empathetic
- Focus on action and movement
- Human-centered approach
- Avoid jargon

## Tagline
